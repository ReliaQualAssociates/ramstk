Reliability Similar Item Assessment Module.

ramstk.analyses.SimilarItem.calculate_topic_633(environment, quality, temperature, hazard_rate)[source]¶

Calculate the Similar Item analysis using Topic 6.3.3 approach.

This method calculates the new hazard rate using the approach found in The Reliability Toolkit: Commercial Practices Edition, Topic 6.3.3.

Parameters

  • environment (dict) – the active environment ID for the from and to environments.

  • quality (dict) – the quality level ID for the from and to quality.

  • temperature (dict) – the aambient operating temperature (in C) for the from and to temperatures.

  • hazard_rate (float) – the current hazard rate of the hardware item being calculated.

Returns

(_change_factor_1, _change_factor_2, _change_factor_3, _result_1); the three change factors (quality, environment, and temperature) and the assessment result.

Return type

tuple

Raise

KeyError if passed an environment, quality, or temperature dict that is missing the from, to, or both keys.

Raise

TypeError if passed a string value for either temperature.
