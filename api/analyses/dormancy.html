Dormancy¶

Dormancy Calculations Module.

ramstk.analyses.Dormancy.do_calculate_dormant_hazard_rate(category_id, subcategory_id, environment_active_id, environment_dormant_id, hazard_rate_active)[source]¶

Calculate the dormant hazard rate for a hardware item.

Attention

The semiconductor category will return a list of two dormancy multipliers. The value at index 0 is for diodes and the value at index 1 is for transistors.

Important

The calling object is responsible for handling any exceptions raised by or passed through this method.

Note

All conversion factors come from Reliability Toolkit: Commercial Practices Edition, Section 6.3.4, Table 6.3.4-1.

Parameters

  • category_id (int) – the component category ID; used as index 0.

  • subcategory_id (int) – the component subcategory ID; used as index 3 for semiconductors.

  • environment_active_id (int) – the active environment ID per the table above; used as index 1.

  • environment_dormant_id (int) – the dormant (storage) environment ID per the table above; used as index 2.

  • hazard_rate_active (float) – the active hazard rate of the component.

Returns

_hr_dormant; the calculated dormant hazard rate.

float

Raise

IndexError if an indexing argument asks for a non-existent index.
