ramstk.analyses.Allocation._calculate_agree_apportionment(time_i, weight_factor, n_sub_systems, n_sub_elements, parent_goal)[source]¶

Perform an AGREE apportionment of a reliability requirement.

Note

the AGREE method uses MTBF as the parent goal.

Parameters

  • time_i (float) – the mission time for the subsystem being calculated.

  • weight_factor (float) – the importance factor for the sub-system currently being allocated.

  • n_sub_systems (int) – the total number of sub-elements in all the sub-systems comprising the hardware item to be allocated.

  • n_sub_elements (int) – the number of sub-elements in the sub-system currently being allocated.

  • parent_goal (float) – the reliability goal of the parent hardware item.

Returns

(_mtbf_alloc, _hazard_rate_alloc, _reliability_alloc); the allocated values for each reliability measure.

Return type

tuple

Raise

TypeError if passed a string for any argument.
