Stress Analysis¶

Component Stress Calculations Module.

ramstk.analyses.Stress.calculate_stress_ratio(stress_operating, stress_rated)[source]¶

Calculate the operating electrical stress ratio of a device.

Inputs can be floats, integers, or a combination.

>>> calculate_stress_ratio(0.382, 1.29)
0.2961240310077519
>>> calculate_stress_ratio(1, 2)
0.5
>>> calculate_stress_ratio(0.382, 2)
0.191
>>> calculate_stress_ratio(1, 1.29)
0.7751937984496123

Rated stress must not be zero: >>> calculate_stress_ratio(0.382, 0.o) Traceback (most recent call last):

…

ZeroDivisionError: float division by zero

Stress inputs must not be strings: >>> calculate_stress_ratio(0.382, ‘3.2’) Traceback (most recent call last):

…

TypeError: unsupported operand type(s) for /: ‘float’ and ‘str’

Parameters

  • stress_operating (float) – the device’s operating level of the stress.

  • stress_rated (float) – the devices’s rated stress.

Returns

_stress_ratio; the ratio of operating stress to rated stress.

Return type

float

Raise

TypeError if an input value is non-numerical.

Raise

ZeroDivisionError if the rated stress is zero.
