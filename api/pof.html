class ramstk.modules.pof.Model.PhysicsOfFailureDataModel(dao, **kwargs)[source]¶

Bases: ramstk.modules.RAMSTKDataModel.RAMSTKDataModel

Contain the attributes and methods of a PhysicsOfFailure (PoF).

The PhysicsOfFailure data model aggregates the Mechanism, OpLoad, OpStress and TestMethod data models to produce an overall PoF. A Hardware item will consist of one PoF. This is a hierarchical relationship, such as:

Mode 1

|_Mechanism 1.1

|_Load 1.1.1 | | | |_Stress 1.1.1.1s | |_Stress 1.1.1.2s | |_Test 1.1.1.1t | |_Test 1.1.1.2t | |_Load 1.1.2


|_Stress 1.1.2.1s |_Stress 1.1.2.2s |_Test 1.1.2.1t
