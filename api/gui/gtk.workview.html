Display Hardware assessment input attribute data in the RAMSTK Work Book.

The Hardware assessment input view displays all the assessment inputs for the selected Hardware item. This includes, currently, inputs for MIL-HDBK-217FN2 and NSWC-11. The attributes of a Hardware assessment input view are:

Variables

  • _lst_labels (list) – the text to use for the assessment input widget labels.

  • _dic_assessment_input (dict) – dictionary of component-specific AssessmentInputs classes.

  • _hazard_rate_method_id (int) – the ID of the hazard rate method used for Hardware item.

  • cmbActiveEnviron – the operating environment for the hardware item.

  • cmbDormantEnviron – the storage environment for the hardware item.

  • cmbFailureDist – the statistical failure distribution of the hardware item.

  • cmbHRType – the type of reliability assessment for the selected hardware item.

  • cmbHRMethod – the assessment method to use for the selected hardware item.

  • fraDesignRatings – the container to embed the piece part design attributes Gtk.Fised().

  • fraOperatingStress – the container to embed the piece part operating stresses Gtk.Fixed().

  • txtActiveTemp – the ambient temperature in the operating environment.

  • txtAddAdjFactor – an adjustment factor to add to the assessed hazard rate or MTBF.

  • txtDormantTemp – the ambient temperature in the storage environment.

  • txtFailScale – the scale parameter of the statistical failure distribution.

  • txtFailShape – the shape parameter of the statistical failure distribution.

  • txtFailLocation – the location parameter of the statistical failure distribution.

  • txtMultAdjFactor – an adjustment factor to multiply the assessed hazard rate or MTBF by.

  • txtSpecifiedHt – the stated hazard rate.

  • txtSpecifiedHtVar – the variance of the stated hazard rate.

  • txtSpecifiedMTBF – the stated mean time between failure (MTBF).
