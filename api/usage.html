class ramstk.modules.usage.Model.UsageProfileDataModel(dao, **kwargs)[source]¶

Bases: ramstk.modules.RAMSTKDataModel.RAMSTKDataModel

Contain the attributes and methods of a Usage Profile.

Class for Usage Profile data model. This model builds a Usage Profile from the Mission, Phase, and Environment data models. This is a hierarchical relationship, such as:

  • Mission 1
    • Mission Phase 11

      • Environment 111

      • Environment 112

      • Environment 113

    • Mission Phase 12

      • Environment 121

      • Environment 122

  • Mission 2
    • Mission Phase 21

      • Environment 211

      • Environment 212

Provide an interface for the Usage Profile data model and an RAMSTK view.

The Usage Profile controller provides an interface between the Usage Profile data model and an RAMSTK view model. A single Usage Profile controller can control one or more Usage Profile data models.
