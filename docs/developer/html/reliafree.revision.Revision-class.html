source code  
Callback function for handling mouse clicks on the Hardware Object
treeview.

Keyword Arguments:
treeview -- the Hardware Object treeview.
event    -- a gtk.gdk.Event that called this function (the
            important attribute is which mouse button was clicked).
            1 = left
            2 = right
            3 = scrollwheel
            4 = forward
            5 = backward

_treeview_row_changed(self, treeview, path, column)

source code 

Callback function to handle events for the Revision Object TreeView. It is called whenever the Hardware Object TreeView is clicked or a row is activated. It will save the previously selected row in the Revision Object TreeView.

Keyword Arguments: treeview -- the Revision Object gtk.TreeView. path -- the actived row gtk.TreeView path. column -- the actived gtk.TreeViewColumn.
