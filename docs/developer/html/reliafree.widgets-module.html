Module widgets

source code

widgets contains functions for creating, populating, destroying, and interacting with pyGTK widgets. Import this module as _widg in other modules that create, populate, destroy, or interact with pyGTK widgets in the RelKit application.

source code  
Utility function to create Dialog widgets.

Keyword Arguments:
_title_   -- the title text for the Dialog.
_parent_  -- the parent window to associate the Dialog with.  Defaults
             to None.
_flags_   -- the flags that control the operation of the Dialog.
             Defaults to gtk.DIALOG_MODAL and
             gtk.DIALOG_DESTROY_WITH_PARENT.
_buttons_ -- the buttons to display and their response values.
             Defaults to gtk.STOCK_OK <==> gtk.RESPONSE_ACCEPT and
             gtk.STOCK_CANCEL <==> gtk.RESPONSE_REJECT.
