_tree_clicked(self, treeview, event)

source code  
Callback function for handling mouse clicks on the Requirement
Object treeview.

Keyword Arguments:
treeview -- the Requirement Object treeview.
event    -- a gtk.gdk.Event that called this function (the
            important attribute is which mouse button was clicked).
            1 = left
            2 = right
            3 = scrollwheel
            4 = forward
            5 = backward

_treeview_row_changed(self, treeview, path, column)

source code 

Callback function to handle events for the FMEA Object treeview. It is called whenever the FMEA Object treeview is clicked or a row is activated. It will save the previously selected row in the FMEA Object treeview.

Keyword Arguments: treeview -- the Requirement Object gtk.TreeView. path -- the actived row gtk.TreeView path. column -- the actived gtk.TreeViewColumn.
