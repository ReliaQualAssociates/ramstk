Class Mechanical

source code

Mechanical Relay Component Class. Covers specifications MIL-R-5757, MIL-R-6106, MIL-R-19523, and MIL-R-39016.

Hazard Rate Models:

  1. MIL-HDBK-217F, section 13.1
